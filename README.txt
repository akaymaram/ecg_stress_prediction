1. ecg_peak_finder contains the peak finder algorithms and results for the QT samples and driver data.

2. feature_extraction contains the scripts for extracting the features from the found peaks for the driver data.

3. original_paper contains Christopher Ottesen's work and some information about the data he used in his work.

4. tpot_code contains scripts for running tpot on our and Ottesen's datasets.


Notes:
	A. Files such as final_file.csv have been placed in multiple directories to enable running from the command line at the directory. Files with identical names contain the same data.
	B. Each of the above four main folders contains a README.txt inside with more details about their contents.
